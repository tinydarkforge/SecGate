What SecGate actually runs

Five industry-standard open source scanners, one normalized output. No AI in the loop, no telemetry, no account.

Semgrep Gitleaks osv-scanner Trivy npm audit
  • Semgrep — static analysis for code-level vulns: insecure deserialization, SSRF, command injection, etc.
  • Gitleaks — secrets in commits and history (API keys, tokens, private keys)
  • osv-scanner — OSV.dev cross-ecosystem dependency CVE check (Node, Python, Go, Rust, Java)
  • Trivy — container image + filesystem scan for known CVEs
  • npm audit — npm registry vulnerability advisories
  • SecGate normalizes all five into one JSON + Markdown report, dedupes, scores, exits non-zero on CRITICAL/HIGH for CI gating
